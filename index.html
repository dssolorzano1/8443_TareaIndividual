XML

XML, también conocido como Extensible Markup Language, es un tipo de lenguaje que se utiliza como opción predeterminada para intercambiar datos en Internet. Independientemente de los lenguajes de programación, necesitan una herramienta para analizarse a sí mismos y luego expresar los mensajes a las personas de la forma que entienden, y XML funciona como esta herramienta de ayuda.

JSON

JSON son las siglas de JavaScript Object Notation. Alguien lo considera una mejor opción que XML para transferir datos porque JSON requeriría menos codificación. Entonces, el tamaño de JSON puede ser mucho menor. Para aquellas páginas web o servicios que requieren procesar una gran cantidad de datos, JSON es una opción perfecta porque puede funcionar mucho más rápido.

SEMEJANZAS DIFERENCIAS
Ambos formatos son un recurso muy habitual para el trabajo en entornos de integración XML es un lenguaje de maquillaje que se presenta a las personas, mientras que JSON es un formato de código escrito en JavaScript.
Una de las similitudes más evidentes entre estas dos estructuras de datos es que JSON y XML están basadas en texto XML es que el contenido no necesita estar delimitado con comillas, como ocurre con los textos en el formato JSON.
Ambos formatos tienen estructuras jerárquicas en las que cada campo tiene su nombre y valor separados por comas. JSON usa una notación simple, mientras que el XML usa una estructura de tags personalizadas para representar los objetos.
JSON y XML son ampliamente utilizados, por lo que son soportados por muchos lenguajes de programación El formato XML soporta diferentes tipos de datos, entre ellos: imágenes y gráficos, lo que no es posible transmitir en el formato JSON
